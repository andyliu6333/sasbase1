PROC SQL

Use SQL inside SAS for selecting, joining, grouping, and creating reusable tables.

Learning outcomes

  • Write SELECT, WHERE, ORDER BY, and GROUP BY clauses.
  • Create summary tables with calculated fields.
  • Join tables while checking for duplicate records.

Example program

proc sql;
  create table work.class_summary as
  select sex,
         count(*) as n,
         mean(height) as mean_height format=6.2
  from sashelp.class
  group by sex;
quit;

Learning activity

Practice: Create a summary table with one row per group. Include a count and one average, then print the result.
