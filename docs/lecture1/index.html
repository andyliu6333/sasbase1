Learning outcomes

  • Explain the role of the DATA and PROC steps.
  • Identify errors, warnings, and notes in the SAS log.
  • Run a complete SAS program and save your work.

Example program

proc print data=sashelp.class;
run;

Learning activity

Practice: Open SAS Studio and run the example program. Change the title, rerun it, then use the log to confirm that the program completed successfully.

Before the next lesson

Save a copy of your working code. Review the SAS log and make sure there are no unexpected errors or warnings before submitting any work.
