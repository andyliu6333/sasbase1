Learning outcomes

  • Write a DATA step that reads an existing SAS table.
  • Use IF/THEN logic to create categories.
  • Use WHERE or IF correctly to filter observations.

Example program

data work.class_clean;
  set sashelp.class;
  bmi_proxy = weight / height**2 * 703;
  if age >= 14 then age_group = 'Older';
  else age_group = 'Younger';
run;

proc print data=work.class_clean;
run;

Learning activity

Practice: Build a cleaned version of SASHELP.CLASS. Add a derived variable and a two-level category, then verify the result with PROC PRINT.

Before the next lesson
