Learning outcomes

  • Apply a format without changing the underlying value.
  • Use common character, numeric, and date functions.
  • Convert variables safely and check for missing values.

Example program

data work.dates;
  input id date_text : $10.;
  date_value = input(date_text, yymmdd10.);
  format date_value date9.;
datalines;
101 2026-06-27
;
run;

Learning activity

Practice: Import a short date field, create a SAS date value, and display it with a readable format.
